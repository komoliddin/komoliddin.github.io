# PRO PROJECTS — Professional Portfolio & CMS

Современная витрина IT-проектов с глубокой интеграцией GitHub, продвинутой админ-панелью и дизайном в стиле **Glassmorphism**.

## 🚀 Технологии
- **Frontend**: Vue 3 (Composition API), Bootstrap 5, FontAwesome 6.
- **Design**: Glassmorphism (эффект матового стекла), адаптивная темная/светлая тема.
- **Backend (Local)**: Node.js + Express.js (обеспечивает прямое сохранение в JSON и менеджмент файлов).
- **GitHub Integration**: Автоматическое подтягивание данных репозиториев (README, релизы, звезды) с умным кэшированием для обхода лимитов API.

## ✨ Ключевые функции
- **Admin CMS**: Полное управление проектами, категориями, контактами и донатами без правок кода.
- **Smart Image Upload**: Загрузка скриншотов через админку с автоматическим созданием уникальных папок для каждого проекта в `image/`.
- **Deep Filtering**: Поддержка подкатегорий с интерфейсом "аккордеон" в сайдбаре.
- **Unified UI**: Единый стиль шапки и футера, стилизованный скроллбар, кастомные курсоры.
- **Reliability**: Система "анти-кэш" (версионность JSON файлов) и сохранение состояния вкладок в админке.

## 🛠 Быстрый старт

### 1. Запуск сайта
Для разработки используйте **Live Server** (VS Code) или любой локальный веб-сервер.

### 2. Работа в админке
1. Запустите `start_admin_server.bat` (не закрывайте консоль).
2. Перейдите на `http://localhost:3000/login.html`.
3. Пароль: `Admin_123`.
4. Вносите изменения и жмите **«СОХРАНИТЬ ВСЕ»**.

## 📂 Структура проекта
- `projects.json`: База данных всех проектов (локальных и GitHub).
- `categories.json`: Иерархия категорий и подкатегорий.
- `config.json`: Глобальные настройки (контакты, соцсети, реквизиты).
- `server.js`: Локальный сервер для сохранения данных.

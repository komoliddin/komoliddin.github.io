# PRO PROJECTS — Professional Portfolio & CMS (Stable v3.0)

Современная витрина IT-проектов с глубокой интеграцией GitHub, продвинутой админ-панелью и дизайном в стиле **Glassmorphism**.

## 🚀 Технологии
- **Frontend**: Vue 3 (Composition API), Bootstrap 5, FontAwesome 6.
- **Design**: Glassmorphism (эффект матового стекла), адаптивная темная/светлая тема.
- **Animation**: Чистый CSS (без тяжелых внешних библиотек) для максимальной стабильности.
- **Backend (Local)**: Node.js + Express.js (обеспечивает прямое сохранение в JSON и менеджмент файлов).
- **GitHub Integration**: Автоматическое подтягивание данных репозиториев (README, релизы, звезды) с умным кэшированием.

## ✨ Ключевые функции
- **Admin CMS**: Полное управление проектами, категориями, контактами и донатами.
- **Stable Navigation**: Исправлена проблема исчезновения карточек при переключении категорий.
- **Smart Image Upload**: Загрузка скриншотов через админку с автоматическим созданием уникальных папок для каждого проекта в `image/`.
- **Deep Filtering**: Поддержка подкатегорий с интерфейсом "аккордеон" в сайдбаре.
- **Contact Sync**: Экспорт полного контакта (.vcf) со всеми данными (имя, тел, email, сайт, TG).
- **Favicon & Manifest**: Полная поддержка PWA и корректное отображение иконок во всех браузерах.

Для разработки используйте **Live Server** (VS Code) или любой локальный веб-сервер.

1. Запустите `start_admin_server.bat` (не закрывайте консоль).
2. Перейдите на `http://localhost:3000/login.html`.
